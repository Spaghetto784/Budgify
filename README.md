# Budgify

Personal budget tracker for iOS built with SwiftUI, SwiftData & Swift Charts.

## Features

- Track expenses by category
- Set monthly budgets
- Visualize spending with native charts
- Multi-currency support

## Tech Stack

- Swift 5.9+
- SwiftUI
- SwiftData
- Swift Charts
- iOS 17+

## Getting Started

1. Clone the repo
2. Open `Budgify.xcodeproj` in Xcode 15+
3. Run on simulator or device (iOS 17+)
